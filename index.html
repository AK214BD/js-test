Javascript Assignment

3. Answer:-
var: Can be assigned and declared multiple times within its scope: Example: var x = 2. var x = 3. No error, but will print the last value (3)

let: Can be reassigned within its block, but not redeclared: Example: let x = 20. let x = "bird". This will throw an error and print the first value only.

const: Cannot be reassigned after its initial declaration: JavaScript const G = 9.8. G = 9.8. This will throw an error because const cannot be reassigned.
